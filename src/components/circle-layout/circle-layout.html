JavaScript 圆形布局

属性 data-circle-layout-display, data-circle-layout-radius, data-circle-layout-radius-c 和 data-circle-layout-rotation 用于设置显示个数、大小和旋转角度。 支持动态添加元素。 

<div class="uix-circle-layout js-uix-circle-layout" data-circle-layout-display="7" data-circle-layout-radius="180" data-circle-layout-radius-c="130" data-circle-layout-rotation="-45">
    <ul>
        <li><a href="#">1</a></li>
        <li><a href="#">2</a></li>
        <li><a href="#">3</a></li>
        <li><a href="#">4</a></li>
        <li><a href="#">5</a></li>
        <li><a href="#">6</a></li>
    </ul>
    <div>
        <div>
            <p>这里放内容</p>
        </div>
    </div>
</div>
