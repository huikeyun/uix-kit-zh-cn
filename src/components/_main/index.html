如何使用？

你电脑上需要安装Node。 用于将构建的文件输出到./dist/* 和 ./examples/*.html.

步骤 1. 使用 NPM（找到您当前的项目目录，然后输入以下命令。）或从以下网址下载最新版本 Gitee或Github。 对于 nodejs，您必须安装一些依赖项。 

sudo yarn uix-kit

或者克隆 repo 以获取所有源文件，包括构建脚本： 

git clone git@gitee.com:huikeyun/uix-kit.git
git clone git://github.com/xizon/uix-kit.git

第 2 步。 首先，使用绝对路径进入您的 "uix-kit/"文件夹目录。

cd /{your_directory}/uix-kit

第 3 步。 在进行所有开发工作之前，请确保您已安装“Node 10+”。 之后，在主目录中运行以下代码来安装节点模块依赖项。 

sudo yarn

第 4 步。 当您准备好部署到生产环境时，创建一个压缩包，其中包含： 

yarn build

完成后，这将启动一个可以在以下位置访问的服务器 http://localhost:8080/examples/

步骤 5. 要在不编译打包的情况下预览页面，请运行以下命令： 

yarn preview

所有页面都可以通过http://localhost:3000

预览
但是没有办法在页面上运行 PHP 或获取 AJAX 请求。 您需要访问服务器上的链接，或通过 MAMP 构建服务器。 然后访问一个类似的 URL： http://localhost:{port}/uix-kit/examples/
